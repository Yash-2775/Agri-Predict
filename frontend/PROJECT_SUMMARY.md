## 🚀 How to Run (3 Steps)

### Step 1: Install Dependencies
```bash
cd agri-predict-app
npm install
```

### Step 2: Start Development Server
```bash
npm start
```

### Step 3: Run on Your Phone
1. Install **Expo Go** app on your phone
2. Scan the QR code
3. Wait for app to load
4. Done! 🎉
